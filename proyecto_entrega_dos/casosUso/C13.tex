\item[3.]El Usuario Registrado ingresa el nombre y selecciona el lugar donde almacenará el documento.

\multicolumn{2}{|p{0.45\textwidth}|}

\begin{itemize}

\item[4.]El Sistema almacena el archivo.

\end{itemize}
